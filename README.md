# E-Commerce Sales & Customer Insights

## Project Overview
End-to-end data analysis of Superstore dataset to uncover sales trends, customer behavior,
profitability drivers, and discount impact using Python, SQL, and Power BI.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (MySQL/PostgreSQL)
- Power BI

## Workflow
1. Data Cleaning & Feature Engineering
2. Exploratory Data Analysis
3. SQL Business Queries
4. Dashboard Creation
5. Insights & Recommendations

## Key Insights
- High discounts reduce profitability.
- Technology category generates highest profit.
- Top customers contribute major revenue share.
- Regional profit margins vary significantly.
